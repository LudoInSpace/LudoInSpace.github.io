xtfa.nodes module#

This class represent the model of a node, as made of an input pipeline (an aggregate computation pipeline), a contention pipeline (a delay-bound pipeline) and an output pipeline (a flow-state computation pipeline). It also stores the FlowStates of the flow at various observation points
